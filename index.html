Activity

What is Activity?

Activity is a PHP API that allows you to log user activity without exposing database credentials.

Demo:

You can access https://demo.miguelpiedrafita.com/Activity with a POST request. NOTE: In the demo, all the data is deleted periodically. In the script, this doesn't happen.

Requirements:

- PHP 5.5.9 or higher

Installation:

  1. Download lastest release from here.
  2. Upload all the files to your server.
  3. Edit site name, timezone, site URL, site email and language at includes/config.php.
  4. Access the script and create an account using the register form.
  5. OPTIONAL: If you want a private install, change $signupstatus = true; to $signupstatus = true; to disable the signup form.
  6. Enjoy!

Support:

  • If you have any problem while installing/using the script, open an issue on my support system.
  • If you find any error in the code, open an issue or, if you know how to solve it, make a pull request.
  • If you have new ideas for this script, go ahead and post them in MP Feedback, under the Activity section.
